# 🧭 一、网站新定位（先定“灵魂”）

你的新网站不再是：

> ❌ 个人主页 / 作品集

而是：

> ✅ 一个独立开发者的「小工具发布与构建中心」

------

## ✨ 新定位一句话（放首页）

> A collection of small tools built to solve everyday problems.

或中文：

> 一个持续发布小工具的构建中心。

------

# 🏠 二、网站整体结构（产品发布站）

```
HOME（首页 / 产品总览）
├── Products（产品列表）
│   ├── BatchFlow（文件工具）
│   ├── PromptNest（AI工具）
│   ├── BuildNote（记录工具）
│
├── Updates（更新日志）
│   每次改动 / 发布记录
│
├── About（关于构建者）
│   你是谁 + 你在做什么
│
└── Now（正在做什么）
    当前开发中的项目
```

------

# 🧱 三、首页（核心页面设计）

首页直接变成“产品发布页”，不是简历。

------

## 🟦 HERO 区（产品站风格）

```
构建日志 / Build Log

A small product hub for useful tools.

I build small tools that solve everyday problems.
```

👇 按钮：

- View Products
- Latest Update
- About

------

# 🧰 四、Products 区（核心）

👉 这是你网站的“主菜”

------

## 🧩 产品卡片结构（统一模板）

每个产品都用这个：

```
产品名称
一句话定位（非常关键）

状态：MVP / 实验 / 发布中

[View Details →]
```

------

## 📦 示例（帮你改好）

------

### 🧰 BatchFlow

```
BatchFlow
Batch file organization tool

A lightweight tool for batch renaming, sorting, and organizing files.

Status: MVP
```

------

### 🤖 PromptNest

```
PromptNest
AI prompt management tool

Organize, categorize, and reuse your AI prompts efficiently.

Status: Experimental
```

------

### 📝 BuildNote

```
BuildNote
Build process logging tool

A simple system to record ideas, progress, and development notes.

Status: In use
```

------

# 🆕 五、Updates（产品站灵魂）

👉 这是你和普通作品集的最大区别

------

## ✨ 页面标题：

> Updates / Changelog

------

## 📌 示例内容：

```
2026.06
- Added PromptNest first version
- Improved file batch processing rules

2026.05
- Started BuildNote system
- Designed product page structure
```

------

👉 这个区域的意义：

> 让别人看到你“持续在做”

比作品更重要
